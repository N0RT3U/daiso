# Data Notes

Raw data and intermediate artifacts are excluded from GitHub.

## Excluded Items
- raw crawling CSV files
- OCR intermediate outputs
- BigQuery loading intermediates
- large GIS spatial files
- presentation images and PDF assets

## Reproduction Guide
- source code lives in `src/`
- analysis notebooks live in `notebooks/`
- reports and design documents live in `docs/`
- schema and derived-variable definitions are documented under `docs/project/`
