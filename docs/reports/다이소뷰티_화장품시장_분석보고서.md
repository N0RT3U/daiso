## 1. 분석 개요

### 1.1 분석 목적

다이소 뷰티의 지속 가능한 성장을 위해 **서울시 화장품 시장**과 **외국인 밀집지역**을 결합 분석하여, 최적의 상권 전략(Hub & Spoke)을 수립한다.

| 다이소 리뷰      | 다이소몰 크롤링       | 2024~2026년               | 323,493개 리뷰             |
| 서울 화장품 매출 | 서울시 상권분석서비스 | 2024년 1분기~2025년 3분기 | 3,515건 (575개 상권)       |
| 외국인 생활인구  | 서울 열린데이터광장   | 2024년 1분기~2025년 3분기 | 25개 자치구                |

## 5. Hub & Spoke 전략 제언

### 5.1 전략 개요

> **핵심 발견:** 복합점수 기반 분류에서 외국인 밀집도 + S-DoT 유동인구 복합점수 상위 30%(0.60점 이상)를 Hub로 분류한다.
> 중구(1.89), 강남구(1.46), 마포구(0.81)가 외국인 관광 Hub의 핵심 거점이다.

```
┌─────────────────────────────────────────────────────────────┐
│                    Hub & Spoke 모델                         │
├─────────────────────────────────────────────────────────────┤
│  HUB (관광 상권)          │  SPOKE (거주 상권)              │
│  • 복합점수 상위 30%       │  • 복합점수 하위 70%             │
│  • 외국인 밀집 + 유동인구  │  • 주택가 인근 상권              │
│  • 연착륙 스킨케어 집중    │  • 다양한 제품 구색              │
│  • 재고 심도 5~10배       │  • 테스트 환경 제공              │
│  • 다국어 서비스          │  • 정기 구매 유도                │
└─────────────────────────────────────────────────────────────┘
```

### 6.1 시장 기회

| 여성 65%, 30~40대 44%        | 타겟이 명확하다                           |
| 14~17시 피크타임             | 해당 시간대에 재고 관리를 집중해야 한다   |

### 6.2 실행 제언

#### 6.2.1 단기 (1~3개월)

1. **명동 매장 집중 강화**

   - VT 리들샷 시리즈 재고 5배 확보
   - 중국어/일본어/영어 POP 설치
   - 14~17시 추가 인력 배치
2. **연착륙 TOP 10 제품 전면 진열**

   - 입구 진열대 최우선 배치
   - "재구매율 60%" 스티커 부착

#### 6.2.2 중기 (3~6개월)

3. **HUB 상권 확대**

   - 강남역, 신사역, 학동사거리 동일 전략 적용
   - 외국인 밀집 시간대 모니터링
4. **SPOKE 상권 구색 최적화**

   - 성수동, 송리단길: 트렌디 신제품 테스트
   - 양천향교: 40대+ 타겟 안티에이징 강화

#### 6.2.3 장기 (6개월~)

5. **데이터 기반 재고 관리**
   - 상권별 매출 데이터 연동
   - 외국인 생활인구 실시간 모니터링
   - 계절/시간대별 재고 자동 조정
